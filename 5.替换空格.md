# 5.替换空格

请实现一个函数，把字符串 s 中的每个空格替换成"%20"。

 

示例 1：

输入：s = "We are happy."
输出："We%20are%20happy."


限制：

0 <= s 的长度 <= 10000



```C++
class Solution {
public:
    string replaceSpace(string s) {
        string res;
        for(int i = 0; i < s.size(); i++)
        {
            if(s[i] != ' ')
            {
                res += s[i];
            }
            if(s[i] == ' ')
            {
                res += "%20";
            }
        }
        return res;

    }
};
```
